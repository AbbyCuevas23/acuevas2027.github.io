Game Inspiration

The game is inspired by the previous game "DahlBate" of one of the developers, Raina Rayelle S. Bautista. DalhBate is a C++ game of 8-Dahlia. If Trash Talk talks about having a trash can in Arayat Classroom, DalhBate is about how Raina's previous section, Dahlia, debates if there's a formative assessment or not. After brainstorming for a game, DalhBate came to Raina's mind, and the developers decided to innovate and improve this game with CSS, HTML, and JavaScript.
